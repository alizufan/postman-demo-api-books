# Node.js - Book Rest-API

Simple Node.js + Vercel Serverless example Book's Rest-API.

## How to Use

You can choose from one of the following two methods to use this repository:

### One-Click Deploy

Deploy the example using [Vercel](https://vercel.com?utm_source=github&utm_medium=readme&utm_campaign=vercel-examples):

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/git/external?repository-url=https://github.com/xboykun/postman-demo-api-books&project-name=api-books&repository-name=api-books)

### Clone and Deploy

```bash
git clone https://github.com/xboykun/postman-demo-api-books
```

Install the Vercel CLI:

```bash
npm i -g vercel
```

Then run the app at the root of the repository:

```bash
vercel dev
```
